# Differential Methylation Analysis
Code for analysis of differntial methylation from the methylation frequencies generated by f5c
I used a combination of [NanoMethViz](https://github.com/Shians/NanoMethViz) and [DSS](https://www.bioconductor.org/packages/release/bioc/vignettes/DSS/inst/doc/DSS.html) in RStudio

## R Code for Differential Methylation

#### Open Libraries
`library(DSS)
library(NanoMethViz)
library(dplyr)`

#### place needed files in a list
`file_list=list.files(path="./Projects/West_vs_East/BothCoastData/MethCall", pattern = "*.tsv", full.names = TRUE)`

`meth_tabix<-file.path(tempdir(), "meth_data.bgz")
samples<-c("NW2","NW3","NW4","NW6","NW7","NWA",
           "SSA","SSE","SSF",
           "MA1","MA2","MA3","MA4","MA5","MA6","MA7","MA8",
           "ME1","ME2","ME3","ME4","ME5","ME6","ME7","ME8",

           "SSB","SSC","SSD",
           "NH1","NH2","NH3","NH4","NH5","NH6","NH7","NH8")`


#### create necessary file format for NanoMethViz
`create_tabix_file(file_list,meth_tabix,samples)`

#### Add Filter/cut off to match methfreq tables generated by nanopolish
`filter.methtab<-paste0(tempfile(),".tsv.bgz")
filter_methy(meth_tabix, output_file = filter.methtab, statistic < -2 | statistic > 2)`

`methy(meth_tabix)<-filter.methtab
methy(meth_tabix)
#convert data format to bisulfite-like data
bss <- methy_to_bsseq(meth_tabix)
bss.f <- methy_to_bsseq(filter.methtab)`

#### Identify DML for each Pairwise
